Задание:

1) Попасть в систему без пароля несколькими способами
2) Установить систему с LVM, после чего переименовать VG
3) Добавить модуль в initrd
\*
4) Сконфигурировать систему без отдельного раздела с /boot, а только с LVM. 
PV необходимо инициализировать с параметром --bootloaderareasize 1m 
---

Решение:
---

## 1. Попасть в систему без пароля несколькими способами ##

**Способ 1**
1.1.1. При запуске ОС, нажимаем **e**, попадая в окно для редактирования загрузки
1.1.2. Удаляем в строке, начинающейся с linux16 опции **console=tty0** и  **console=ttyS0,115200n8**
1.1.3. Добавляем **init=/bin/sh** и нажимаем **ctrl-x**
1.1.4. Попадаем в систему с приглашением **sh-4.2#**
1.1.5. Аналогично если бы в системе мы выполнили бы **chroot /sysroot**
1.1.6. Смотрим что у нас примонтировано командой **mount | less** и самая верхняя строка выглядит - **sysfs on /sys type sysfs (rw,nosuid,nodev,noexec,relatime)**
1.1.7. Смотрим **id** и видим **uid=0(root) gid=0(root) groups=0(root)**
1.1.8. Смотрим **mount -t xfs** видим что система подключена в режиме RO **/dev/sda1 on / type xfs (ro,relatime,attr2,inode64,noquota)**
1.1.9. Чтобы примонтировать в режим RW (для редактирования) необходимо выполнить **mount -o remount,rw /** после чего можно проверить **mount -t xfs**
1.1.10. Дальше уже можно скидывать пароль рута, если Selinux включен, то необходимо будет создать файлик в корне **/.autorelabel** чтоб Selinux 

**Способ 2**
1.2.1. Выполняем шаги из п. 1.1.1 по 1.1.2
1.2.2. Добавляем **rd.break** и нажимаем **ctrl-x**
1.2.3. Попадаем в систему с приглашением **switch_root:/#**
1.2.4. Смотрим что у нас примонтировано командой **mount | less** и самая верхняя строка выглядит - **rootfs on / type rootfs (rw)**
1.2.5. Смотрим **mount -t xfs** видим что система подключена в режиме RO **/dev/sda1 on / type xfs (ro,relatime,attr2,inode64,noquota)**
1.2.6. Чтобы примонтировать в режим RW (для редактирования) необходимо выполнить **mount -o remount,rw /** после чего можно проверить **mount -t xfs**
1.2.7. Чтоб скинуть пароль необходимо выполнить команду **chroot /sysroot**
1.2.8. **passwd root** - меняем пароль, дальше создаем файлик **/.autorelabel** командой **touch /.autorelabel** 
1.2.9. Выходим из **chroot** командой **exit**
1.2.10. Перемонтируем в режим RO **mount -o remount,ro /** и перегружаемся

**Способ 3**
1.3.1. Выполняем шаги из п. 1.1.1 по 1.1.2
1.3.2. Заменяем **ro** на **rw init=/sysroot/bin/sh** и нажимаем **ctrl-x**
1.3.3. Попадаем в систему с приглашением **:/#**
1.3.4. Смотрим что у нас примонтировано командой **mount | less** и самая верхняя строка выглядит - **rootfs on / type rootfs (rw)**
1.3.5. Смотрим **mount -t xfs** видим что система подключена в режиме RO **/dev/sda1 on /sysroot type xfs (rw,relatime,attr2,inode64,noquota)**
1.3.6. Видим что ФС сразу доступна на редактирование (**RW**)
1.3.7. Далее можно менять пароль рута...
